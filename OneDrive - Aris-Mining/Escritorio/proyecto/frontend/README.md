# Frontend - Creación de Usuarios Operativos Microsoft 365

Front-end en React + TypeScript para la creación de usuarios operativos integrado con Microsoft 365.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación se ejecutará en `http://localhost:3000`

## Build

```bash
npm run build
```

## Variables de Entorno

Crear un archivo `.env` en la raíz del proyecto:

```
VITE_API_BASE_URL=http://localhost:5000
```
